الدورات التي يقدمها المدرب رياض ثلجي

دورات إدارية

دورة التحضيرلامتحان مديرالمشروع المحترف PMP

تقوم هذه الدورة بإعداد المشاركين لامتحان إدارة المشاريع الاحترافية PMPوتحتوي على ورش عمل مهنية وأسئلة تدريبية مع تحليل متعمق، ويعتبر برنامجنا أحد أهم البرامج في هذا المجال في المنطقة.

دورة إدارة المشاريع الرشيقة بأسلوب Scrum وأجايل

هذه الدورة مقدمة لأساليب إدارة المشاريع الرشيقة بتقنية أجايل وسكرم.

دورة Primavera P6

Primavera هي أفضل أداة برمجية لجدولة إدارة المشاريع ، حيث تمنح هذه الدورة المشاركين كل ما يحتاجون إليه للبدء بثقة في استخدام Primavera في مجال التطبيق الواقعي.

دورة Microsoft Project

تبني هذه الدورة مهارات المشاركين في جدولة المشاريع باستخدام أداة برنامج Microsoft Project لأي نوع من المشاريع.

دورة تدريب المدربين

دورة تدريب المدربين أو الـ TOT تنمي مهارات المدربين أوالطامحين للعمل في مجال التدريب بالمهارات والتقنيات اللازمة لجعلهم مميزين في هذا المجال.

دورة التسويق والمبيعات

هذه الدورة تبنى وتنمي مهارات التسويق والبيع لدى المشاركين، وتربطها بأحدث البرمجيات المتعلقة بهذا المجال مثل Salesforce.

دورات تكنولوجيا المعلومات

دورة تطوير الأنظمة باستخدام Oracle

هذه الدورة تبني لدى المشاركين المهارات اللازمة لتطويرالانظمة بتقنية أوراكل بشكل محترف.

أمثلة من المشاركين: بنك الأمة الليبي، وشركة تطبيقات المستقبل للكومبيوتر.

دورة برمجة SQL Server

هذه الدورة تبني مهارات برمجة قاعدة البيانات SQL Server بلغة SQL و T-SQL بشكل محترف وعملي بنسبة 90 بالمئة.

أمثلة من المشاركين: شركة نقل الكهرباء العمانية

دورة تطوير مواقع الانترنت باستخدام HTML و CSS و جافا سكربت

هذه الدورة تؤسس وتطور مهارات المشاركي في بناء مواقع الانترنت باستخدام لغات HTML, CSS و JavaScript

وتم تقديمها عدة مرات كدورة جماهيرية.

تطوير مواقع إنترنت متقدمة بتقنية ASP.Net

هذه الدورة تبني مهارات احترافية في برمجة مواقع الانترنت من طرف السيرفر، بتقنية ASP.Net من ميكروسوفت.

وقد تم تقديمها عدة مرات كدورات جماهيرية شارك بها مرشحون من البنك العربي، وشركة الكهرباء الاردنية، وشركة طلال أبو غزالة.

دورات البرمجة بلغة #C و++C و جافا بشكل عملي

كل من هذه الدورات تبني لدى المشاركين المهارات الاحترافية اللازمة لبناء تطبيقات ميزة في هذه اللغات، ونسبة التطبيق العملي فيها 90 بالمئة.

أمثة من الزبائن: شركة تطبيقات المستقبل للكومبيوتر

دورة برمجة مواقع التجارة الالكترونية E-Commerce باستخدام ASP.Net

هذه الدورة تبني بشكل عملي لدى المشاركين المهارات البرمجية الاحترافية لبناء مواقع وتطبيقات التجارة الالكترونية باستخدام تقنية ASP.Net وهي من أفضل التجارب لدى المشاركين.

وتم تقديمها عدة مرات على شكل دورات جماهيرية، شارك فيها مشاركون مميزن، فرديون ومن جهات مميزة كالبنك العربي.

دورة SCADA

هذه الدورة تبني وتنمي بشكل عملي واحترافي مهارات المشاركين في برمجة أنظمة التحكم الآلي سكادا، بتقنية SQL Server.  كما تركز مفاهيم السكادا، وربطها في المجال العملي.  ويتم تقديمها على نسخة معدلة من بيانات الشركة الحقيقية. 

مثال على المشاركين: شركة نقل الكهرباء العمانية.
